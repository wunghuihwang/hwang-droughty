# W-Community (황씨가문 사이트)

심심해서 만들어본 개인 프로젝트입니다.  
황씨 가문의 커뮤니티 느낌을 살려 간단한 게시판 및 알림 기능을 구현했습니다.

## 기능

- 로그인, 회원가입 기능 _ 완료
- 게시글 작성, 수정, 삭제 _ 완료
- 댓글 작성 및 조회 _ 미완
- 좋아요 및 북마크 기능 _ 미완
- 알림(Notification) 시스템 _ 미완

## 기술 스택

- **프론트엔드:** Next.js 16, React 19, TypeScript
- **스타일링:** MUI, Emotion
- **상태 관리:** Zustand
- **데이터/백엔드 연동:** Supabase, React Query
- **유틸:** dayjs

## 설치 및 실행

git clone https://github.com/wunghuihwang/w-community.git
cd w-community
